# ClickUp Ticket - Raycast Extension

Quickly create ClickUp tickets directly from Raycast.

## Features

- Create tickets with just a title
- No UI needed - type and press Enter
- Configurable List ID and API token

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run in development mode:
   ```bash
   npm run dev
   ```

3. Configure preferences in Raycast:
   - **ClickUp API Token**: Get from ClickUp Settings → Apps → API Token
   - **List ID**: The ID of the list where tickets will be created

## Usage

1. Open Raycast
2. Type "Create ClickUp Ticket"
3. Enter the ticket title
4. Press Enter

## Development

```bash
npm run dev    # Start development mode
npm run build  # Build for production
npm run lint   # Run linter
```
